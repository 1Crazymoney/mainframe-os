Building Mainframe Apps

You can build a Mainframe app using any Web technology supported by Electron v2 (Chromium v61).

The Mainframe teams build apps using React Native Web. You can see a sample setup in the reference apps, Onyx Stats and create-mainframe-dapp.

Morpheus UI

Morpheus UI is a React Native Web UI toolkit to make it easier for developers to build Mainframe app interfaces.

Available packages include @morpheus-ui/core, @morpheus-ui/fonts, @morpheus-ui/forms, and @morpheus-ui/icons.

Install from NPM

Sample Morpheus UI Usage:

import React from 'react'
import ReactDOM from 'react-dom'
import { Button } from '@morpheus-ui/core'

//Import the fonts in your root component
import '@morpheus-ui/fonts'

function App() {
  return <Button title="Hello World" />
}

ReactDOM.render(<App />, document.querySelector('#app'))

Mainframe SDK

Your app will need to use the Mainframe SDK to interact with the daemon and get access to user data, blockchain interfaces, and integrated storage and communication services. You can view the available API's in the SDK package.

Install from NPM

npm install @mainframe/sdk

npm docs

Example SDK usage:

import MainframeSDK from '@mainframe/sdk'
const sdk = new MainframeSDK()
const res = await sdk.apiVersion()

create-mainframe-dapp

Use create-mainframe-dapp for easy setup with built-in integration with Morpheus UI and Mainframe SDK.
