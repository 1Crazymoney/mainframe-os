# Application SDK

The [Mainframe SDK](https://github.com/MainframeHQ/js-mainframe/tree/master/packages/sdk) is a JavaScript library that can be used by applications in order to interact with APIs provided by the platform.

## APIs

The APIs are updated frequently as features are being built. For an up-to-date version, check out the [`@mainframe/sdk` library implementation](https://github.com/MainframeHQ/js-tools/tree/master/packages/sdk).
